# Automatic Glossary Generator

## Описание проекта

**Automatic Glossary Generator** - это проект для автоматического составления терминов и определений на основе програмнных и отчетных документов. 

### Предварительные требования

- **Python 3** для выполнения основного кода приложения и управления зависимостями
- **Git** для клонирования репозитория
- **Ollama** для запуска языковой модели

### Установка и запуск на Windows (с помощью инсталлятора)

1. **Установка и скачивание необходимых программ**:
- Python версии 3.10+ : https://www.python.org/downloads/ 
- Git: https://git-scm.com/install/windows
- Ollama: https://ollama.com/download/windows


2. **Клонирование репозитория:**

Открыть консоль и ввести команду:
```
git clone https://github.com/MrSmith2/automatic-glossary-generator
```

3. **Установка приложения**

Перейти в директорию с приложением и запустить установщик install.bat, дождаться окончания установки

4. **Запуск приложения**

После успешной установки запустить run.bat
